# MathView — 7th Grade Intervention Dashboard

A data-driven intervention tracking dashboard for 7th grade math teachers.

## Features (Sprint M1)
- **Overview**: iReady score distribution, domain gap charts, tier breakdown by period
- **Roster**: 94 students across Periods 2–5 with tier badges, domain profiles, tag editor
- **Student Profiles**: Domain bars, qualitative tag bank (7 categories), notes, intervention recommendations
- **CSV Import**: Drag-and-drop with auto column mapping
- **Persistence**: localStorage save/load/export

## Planned Sprints
- **M2**: Growth trends, Qualitative Data view, Group Builder, Interventions
- **M3**: Lesson Plans (12 plans, CTE contexts, IEP/MLL/Advanced adaptations)
- **M4**: Math Games (18 games with guide modals), Reports, Print CSS

## Deploy
Deployed via Railway — auto-deploys from `main` branch.

## Stack
- Node.js + Express (backend)
- Vanilla JS SPA (frontend)
- Chart.js 4.4.1
